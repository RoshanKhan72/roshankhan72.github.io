# Mohammad Roshan Soudagar - Portfolio

A modern, responsive portfolio website built with HTML, CSS, and JavaScript for Mohammad Roshan Soudagar, a B.Tech Information Science Engineering student.

## 🚀 Quick Start

### 🌐 **Live Demo**
**GitHub Pages**: https://roshankhan72.github.io

### 📱 **Features**
- **Responsive Design**: Fully responsive layout that works perfectly on desktop, tablet, and mobile devices
- **Dark/Light Mode**: Toggle between dark and light themes with smooth transitions
- **Smooth Animations**: Modern animations and micro-interactions throughout the site
- **Interactive Navigation**: Sticky navigation with smooth scrolling and active link highlighting
- **Contact Form**: Functional contact form with validation and user feedback
- **SEO Optimized**: Meta tags, sitemap, and robots.txt for Google indexing

## 🛠️ Technologies Used

### Frontend
- **HTML5**: Semantic markup for accessibility and SEO
- **CSS3**: Modern CSS with custom properties, Grid, Flexbox, and animations
- **JavaScript (ES6+)**: Modern JavaScript with DOM manipulation and event handling

### Design & Styling
- **CSS Variables**: Custom properties for easy theming and maintenance
- **Responsive Grid System**: Modern layout system for complex designs
- **Modern Typography**: Inter and JetBrains Mono fonts for professional appearance
- **Font Awesome**: Icon library for social media and UI icons
- **Google Fonts**: Web fonts for optimal typography

### Features Implementation
- **LocalStorage**: Theme preference persistence
- **Intersection Observer API**: For scroll-triggered animations
- **Form Validation**: Client-side form validation with user feedback
- **Debouncing**: Performance optimization for scroll events

## 🚀 Deployment

### 🌐 **Live URL**: https://roshankhan72.github.io

### 📋 **Deployment Options**:

#### **GitHub Pages (Recommended - Free)**
1. Create repository: `roshankhan72.github.io`
2. Upload all files to repository
3. Enable Pages from main branch
4. Site goes live at: `https://roshankhan72.github.io`

#### **Netlify (Alternative - Free)**
1. Go to https://app.netlify.com/drop
2. Drag and drop Portfolio folder
3. Get instant live site with custom URL
